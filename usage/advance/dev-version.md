# DEV版本应用

PotatoVN 会在每次提交代码时[自动编译](https://github.com/GoldenPotato137/PotatoVN/actions/workflows/build.yml)一个DEV版本的应用，供有兴趣的用户测试最新功能与修复。

该DEV版本应用是使用自签名证书签名的，在第一次安装时，需要手动信任该证书，具体操作为：
1. 双击压缩包里的cer文件，打开证书导入向导
2. 点击左下角的安装证书，存储位置选择本地计算机，点击下一步
3. 在下个界面中选择“将所有的证书放入下列存储”，点击浏览
3. 选择“受信任的根证书颁发机构”，点击确定，点击下一步，完成导入

该证书只需要导入一次，之后安装DEV版本应用时直接双击msix文件安装即可。


DEV版本的应用是独立与测载版与微软商店版的，安装后会生成一个新的应用数据文件夹，数据不会与其他版本冲突。当你不需要它时，放心将其卸载即可，不会影响其他版本的使用。下图展示了DEV版本应用的图标，和其他版本均不一样，请注意区分。
